# Manila AI Film Festival (MAIFF) Website

Welcome to the official repository for the **Manila AI Film Festival (ai.pinoyseoul.com)** static landing page.

This project is open-source and maintained by **Pinoyseoul Media Enterprise**. 
It serves as the digital front-door for the Philippines' first dedicated festival celebrating films made with artificial intelligence.

## About the Festival
The Manila AI Film Festival connects the Philippines' vast creative workforce with global AI technological infrastructure. It is a 2-day event combining hackathons, AI filmmaking workshops, and film screenings to bridge the gap between traditional industry practices and next-gen AI tools.
